# Stable Diffusion (Local)

This folder manages local Stable Diffusion image generation
used for story scene illustrations.

## Model
- runwayml/stable-diffusion-v1-5
- Loaded via HuggingFace Diffusers
- Zero-cost, runs locally (CPU/GPU)

## Usage
1. Download model
2. Generate images scene-wise
3. Optionally apply LoRA for characters
